Batch Update TFL Summaries

Follow these steps to batch update TFL summary objects.

  1. Navigate to TFL and select TFL Summary.

  2. At the bottom, click Batch Update.

  3. Select the sections where you wish to perform a batch update on TFL summary objects.

  4. Select the specific TFL summary objects you want to update within those sections.

    To update all objects listed, click Select All at the bottom.

  5. Confirm your selections by clicking Confirm.

    The update will proceed in the background, allowing you to continue working on other parts of the document during this time.

  6. Once the batch update of TFL summary objects is complete, a prompt will appear asking you to apply the changes. Click Apply to replace the old TFL summary objects in the document with the updated versions.

tip

After the batch update task is triggered, you can track and manage it in the AI ​​task panel.
