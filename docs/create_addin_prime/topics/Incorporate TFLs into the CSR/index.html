📄️ Insert TFLs Using Placeholders

Instead of directly incorporating tables, figures, and listings into your document, first place TFL placeholders within the relevant sections. Subsequently, execute a data synchronization. This process ensures that as soon as TFL source files become available or receive updates, the latest versions are automatically integrated into the appropriate locations within your document. By adopting this approach, you can strategically arrange for the inclusion of TFLs in your document ahead of their availability. Additionally, this method simplifies the task of maintaining the currency of all in-text TFLs upon their updates, making it an effortless process.

📄️ Prime Create Add-In Overview

Seamlessly integrated with Microsoft Word 365, AuroraPrime Create Add-In is a purpose-built extension designed to elevate the creation of clinical documentation. Tailored specifically for crafting essential clinical documentation like clinical study reports (CSRs) and study protocols, this add-in serves as a powerful tool for medical writers to produce high quality documentation in an efficient manner.

📄️ Publish Template

📄️ Recognize TFLs and TFL Summaries

The Create add-in offers tools for integrating Tables, Figures, and Listings (TFLs) into Microsoft Word documents as specialized TFL objects. These objects allow for direct manipulation and processing by the add-in. However, TFLs that are manually inserted via Word's Insert menu or through third-party tools will initially be regular tables, figures, and listings. These need to be identified and transformed into TFL objects to be compatible with the Create add-in's functionalities. Similarly, any standard text segments serving as summaries for TFLs must also be detected and converted into TFL summary objects. This conversion is essential for the Create add-in to effectively parse and manipulate these elements within the document.

📄️ Remove Variable Instances

Under the Variables tab, each variable is listed alongside a count of its instances, indicating the number of locations it has been inserted into the document. When you expand a variable that appears multiple times, you'll see detailed information about each instance, including the section number and its surrounding context.

📄️ Reset Document Purpose

You can select an action, like writing a document or designing a template, which will take you to the corresponding functionality pane.

📄️ Work with Variables

Integrating variables into your document offers a streamlined approach to managing recurrent pieces of information, such as the name of a test drug or its usual route of administration. By defining these repeating elements as variables, you can insert the relevant variable wherever needed within the document. This method centralizes updates; changing the information in one place automatically reflects across all instances where the variable is used. Utilizing variables enhances the uniformity, standardization, and structure of your document’s wording. This not only significantly reduces the time spent on manually updating repeated information but also minimizes the risk of human errors in editing. Such a strategy is particularly beneficial in documents requiring frequent updates or containing complex, repetitive data, ensuring both efficiency and consistency in document preparation.
