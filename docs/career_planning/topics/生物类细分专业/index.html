生物类细分专业

生物科学（理学学士）

  • 就业范围生物或医学科研院所、大中专学校、技术监督部门、海关检验检疫、食品生产公司与质量监督部门、生物制药公司、农、林科研院所或生产部门、制药公司等。

生物技术（理学学士）

  • 就业范围大中专学校、生物或医学科研院所、环境保护部门、检验与检疫部门、技术监督部门、生物制药公司、食品公司、农、林科研院所或生产部门等。

生物信息学（理学学士）

  • 就业范围国家科技或信息业管理部门、生物信息及相关领域的科研院所、高等学校、生物技术公司、医药开发与生产公司、计算机与信息类高新技术公司。

安全

  • 还有学习化学和生物学的人在实验室要注意保护自己，严格按照操作流程和安全要求进行实验，避免泄漏、起火和爆炸事件的发生。有些国家的大学规定，在化学和生物实验室吃喝东西，是严重的事件，会遭到开除处理的，甚至你拿着饭盒从实验室门口走过都不允许。

科研

  • 生物科学可能是人类投入最多也是最难啃的最后一块硬骨头，从植物、动物和人，都是研究的对象，研究面很宽，目前，需要的是研究人才，尤其是高级研究人才。

就业

  • 目前，生物学或称生物科学的研究还没有形成大的产业群，不像机械、化工、电子等实体经济发达，就业人数庞大。比如就疫苗的研究和开发生产来说，研究的投入很大，需要的研究人员和仪器设备很多也很昂贵，一旦研究出来，生产的量是很少的
  • 从全世界来看，五百强企业中没有一家是生物技术企业，这是由于生物技术产业具有高风险、高收入、高回报和长周期的特点。
  • 生物科学的本科就业前景并不是十分好，除了当老师和继续深造搞科研，几乎没有其他就业方向。选择这个专业的时候一定要慎重。
  • 单纯从理学和工学的就业前景看，理学是不如工学的。所以，如果家长给孩子报考生物学相关专业，一定要配合相应的工学专业来进行选择，这样才能达到就业效果的最大化。

课程

  • 本科阶段只是对生物科学的所有分支专业进行一个初步的系统的学习，其中最重要的专业课当数生物化学。
  • 除了生物化学以外，学生还会在本科期间系统地学习植物学、动物学、生理学、生态学、水生生物学、神经生物学、微生物学、遗传学、发育生物学、细胞生物学、分子生物学、生物信息学等相关知识。

贡献

  • 化学和生物的高端研究的目的和结果是新型药品和人类疾病的治疗方法，因为人类与疾病的斗争是长期和艰巨的任务，学习化学和生物学要有这样的目标。否则，就到低端的食品厂、酿造厂等企业工作，虽然工作无贵贱之分，但贡献和成就有大有小。

生物科学类专业的实验危险吗？​

理工科实验里有危险的也有不危险的

另外一些设备的不正确使用或者老化导致的损坏也会导致危险，前者可以避免，后者看人品。

国内不知道，国外实验室有安全培训的，且安全检查特别勤快

不要在实验室里吃东西，3 m 口罩护目镜隔热手套要常备

每个方向都会有危险的实验吧，但是一般都有配对的防护措施，比如有毒试剂在通风橱操作，只要自己心存敬畏，谨慎即可。
